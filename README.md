# Furniture Store 
![HTML](https://img.shields.io/badge/html5%20-%23E34F26.svg?&style=for-the-badge&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/css3%20-%231572B6.svg?&style=for-the-badge&logo=css3&logoColor=white)
A stylish furniture storefront showcasing products with details, designed using only HTML and CSS.

## ✨ Features

- Hero Section: Attractive design with a background and a call to purchase button.
- Stats Section: Statistics reflecting the store s credibility.
- Product Grid: Displays products with images and prices.
- Image Slider: Displays images of new products.
- Testimonial Section: Customer reviews.
- Subscribe Section: Mailing List Subscription Form.
- Footer: Additional Links and Social Media Accounts.
